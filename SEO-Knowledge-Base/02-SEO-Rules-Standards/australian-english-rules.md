# Australian English Writing Rules — SecurityBlogs Standard

## Overview

All content on SecurityBlogs and for Australian security clients is written in Australian English. This is non-negotiable. Using American English spelling, idioms, or references in Australian-targeted content undermines local SEO signals, reduces trust with Australian readers, and signals to Google that the content is not localised.

---

## Spelling Differences: Australian vs American

### -ise vs -ize
Australian English uses -ise (not -ize) for verb forms.

| American | Australian |
|---|---|
| optimize | optimise |
| analyze | analyse |
| recognize | recognise |
| organize | organise |
| prioritize | prioritise |
| customize | customise |
| maximize | maximise |
| minimize | minimise |
| capitalize | capitalise |
| emphasize | emphasise |
| specialise | specialise ✅ |

### -our vs -or
Australian English retains the -our ending.

| American | Australian |
|---|---|
| color | colour |
| behavior | behaviour |
| favor | favour |
| neighbor | neighbour |
| honor | honour |
| labor | labour |

**Note:** The Australian Labor Party officially uses "Labor" (the American spelling) — this is the exception when referring to the political party specifically.

### -re vs -er
Australian English uses the -re ending for certain words.

| American | Australian |
|---|---|
| center | centre |
| fiber | fibre |
| theater | theatre |
| liter | litre |
| meter (unit) | metre |
| meter (device) | meter ✅ (same) |

### -ence vs -ense
| American | Australian |
|---|---|
| defense | defence |
| offense | offence |
| license (noun) | licence (noun) |
| license (verb) | license (verb) ✅ (same) |

### Double L in Australian English
Australian English doubles the "l" in some verb conjugations where American does not.

| American | Australian |
|---|---|
| traveling | travelling |
| labeled | labelled |
| canceled | cancelled |
| modeling | modelling |
| counselor | counsellor |
| fulfillment | fulfilment (one l) |

### Programme vs Program
- **programme** — a scheduled series of events, a training programme, a government programme
- **program** — a computer program, software application (technical computing context)

### Grey vs Gray
- Australian: **grey**
- American: gray

## Punctuation Differences

### Date Format
Australian: DD/MM/YYYY or D Month YYYY  
Example: 15 January 2025 (not January 15, 2025)

### Time Format
Australian: 9:00am, 3:30pm (lowercase, no space before am/pm)

### Currency
Australian: $150 AUD or AU$150 (not USD or $150 without specifying AUD when context is ambiguous)

### Thousands Separator
Australian: 1,000 | 10,000 | 100,000 (comma separator, same as US)

### Quotation Marks
Australian English uses single quotation marks for standard quotes ('like this'), and double quotation marks for quotes within quotes ("like 'this' example").

However, for digital content (web, markdown), double quotation marks are acceptable and widely used. Consistency matters more than strict preference.

## Australian-Specific Security Industry Terms

| Generic/American Term | Australian Context |
|---|---|
| Burglar alarm | Alarm system / intruder alarm |
| Home alarm company | Security installer / alarm installer |
| Security guard agency | Security guarding company |
| Licensing board | Security Industry Licensing (each state has its own) |
| Background check | Police check / National Police Certificate |

### Australian Regulatory References
When writing about security industry regulations, use:
- **ASIAL** — Australian Security Industry Association Limited (not "the security association")
- **Security Industry Act** — referenced by state (each Australian state has its own)
- **ASIO** — Australian Security Intelligence Organisation (for national security context)
- **AS/NZS 2201** — the Australian standard for intruder alarm systems
- **AS 4806** — CCTV standard series
- **Essential Eight** — ACSC's cybersecurity framework (not "Australian cybersecurity guidelines")
- **SOCI Act** — Security of Critical Infrastructure Act 2018

## Australian Place Names & Geography

Use correct Australian spelling and naming conventions:
- **States:** New South Wales (NSW), Victoria (VIC), Queensland (QLD), South Australia (SA), Western Australia (WA), Tasmania (TAS), Australian Capital Territory (ACT), Northern Territory (NT)
- **Territories:** ACT contains Canberra (the national capital); NT contains Darwin
- **Cities:** Sydney, Melbourne, Brisbane, Perth, Adelaide, Hobart, Canberra, Darwin
- Do not refer to Australia as "down under" in formal content

## AI-Assisted Writing Caveat

AI writing tools (including Claude) default to American English. Every piece of AI-assisted content must be reviewed and edited for Australian English spelling before publication. Use Australian English spell-check in your text editor and run a final find-and-replace for common American variants before publishing.
