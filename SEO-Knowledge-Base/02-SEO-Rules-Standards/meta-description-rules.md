# Meta Description Rules — SecurityBlogs Standard

## Purpose

Meta descriptions appear below the title tag in Google search results. While they are not a direct ranking factor, they are the primary driver of click-through rate (CTR) from the SERP. A strong meta description increases organic traffic without improving rankings.

---

## Character Limits

| Rule | Value |
|---|---|
| Maximum length | 160 characters (including spaces) |
| Minimum length | 120 characters |
| Ideal range | 140–158 characters |

Google truncates meta descriptions beyond approximately 920px in width (~160 characters on desktop, ~120 characters on mobile).

---

## Structure Formula

**[Hook/primary keyword statement]. [Value proposition or proof point]. [Call to action].**

**Example (Service Page):**
"SecurityBlogs delivers specialist SEO for CCTV installers, access control companies, and security SaaS vendors. +180% avg traffic growth. Book a free strategy call."

**Example (Blog Post):**
"Learn how Australian security businesses are using AEO to win featured snippets and appear in AI Overviews — with a step-by-step framework you can apply today."

**Example (Location Page):**
"SecurityBlogs provides security SEO services for businesses across Sydney. We specialise in local rankings for CCTV, alarm, and access control companies. Get a free audit."

---

## Writing Rules

1. **Include the primary keyword naturally** — match user intent
2. **Include a compelling benefit or proof point** — a stat, a specific outcome, or a clear value claim
3. **Include a soft call to action** — "get a free audit," "read the guide," "learn how," "book a call"
4. **Write in active voice** — not passive
5. **Write in Australian English** — no American spelling
6. **Do not use quotes** — Google sometimes strips them and replaces with escaped characters
7. **Each page must have a unique meta description** — no duplicates
8. **Do not keyword-stuff** — one natural mention of the primary keyword is sufficient
9. **Do not start with "We are..."** — Google will rewrite it; start with value
10. **Do not repeat the page title verbatim** — the description should complement the title, not repeat it

## Meta Description by Page Type

### Homepage
Focus on the brand positioning statement — what SecurityBlogs is, who it serves, and the primary outcome.

**Example:** "Australia's only specialist digital marketing agency for the security industry. SEO, AI visibility, Google Ads & web design for CCTV, access control & security SaaS. Get your free audit."

### Service Pages
Lead with the service name, the target client, and the top result metric.

**Example (Security SEO):** "Specialist Security SEO for Australian CCTV installers, access control integrators, and security SaaS vendors. Average +180% organic traffic growth. Book your free audit today."

### Blog Posts
Lead with what the reader will learn and why it matters to them now.

**Example:** "Discover how AEO (AI-Enabled Optimisation) is helping security businesses appear in ChatGPT and Google AI Overviews. Includes a practical 5-step framework for security installers."

### Knowledge Hub Category Pages
Describe the topic cluster and the reader benefit.

**Example:** "SecurityBlogs' Security Industry SEO resource hub. In-depth guides, keyword research templates, and case studies for CCTV, access control, and security SaaS companies."

### Location Pages
Lead with the local service, location, and a local trust signal.

**Example:** "SecurityBlogs delivers security SEO for businesses across Melbourne. We specialise in local rankings for alarm installers, CCTV companies, and access control integrators in Victoria."

### FAQ Pages
Describe the questions answered and the authority behind the answers.

**Example:** "Answers to the most common questions about SEO for security businesses — including costs, timelines, keyword strategies, and AI visibility for CCTV and access control companies."

---

## What Google Will Rewrite

Google rewrites meta descriptions when they:
- Are too short (below 120 characters)
- Don't match the user's search query
- Are keyword-stuffed or spammy
- Repeat the title tag verbatim

Even if Google rewrites your description for some queries, having a well-written description ensures it is used for exact-match and close-match searches.

---

## Meta Description Checklist

Before publishing any page, verify:
- [ ] Description is between 140–158 characters
- [ ] Primary keyword appears naturally in the description
- [ ] Description includes a value proposition or proof point
- [ ] Description includes a call to action
- [ ] Description is unique (no other page has the same description)
- [ ] Written in Australian English
- [ ] Active voice used
- [ ] Does not repeat the title tag verbatim
- [ ] Does not contain quotes
