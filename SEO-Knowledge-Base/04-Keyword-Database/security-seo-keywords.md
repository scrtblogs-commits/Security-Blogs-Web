# Security SEO Keywords — SecurityBlogs Keyword Database

## How to Use This Database

This database contains keyword targets for SecurityBlogs' own website and for Security SEO client campaigns. Keywords are categorised by intent, sub-sector, and geographic modifier. Use these as the starting point for keyword research, content briefs, and on-page optimisation.

**Intent Key:**
- `[I]` = Informational (research/learning intent)
- `[C]` = Commercial investigation (comparison/research before buying)
- `[T]` = Transactional (ready to buy or enquire)
