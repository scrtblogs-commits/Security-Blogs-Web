## KEYWORD PRIORITY MATRIX

Use this matrix to prioritise keyword targets:

| Priority | Criteria | Action |
|---|---|---|
| **P1 — Immediate** | High buyer intent + low competition + directly tied to SecurityBlogs service | Optimise existing page or create new service/landing page immediately |
| **P2 — Short-term** | Medium intent + medium competition + strong revenue connection | Create dedicated blog post or cluster content within 30 days |
| **P3 — Medium-term** | Informational + lower competition + builds topical authority | Add to Knowledge Hub content calendar |
| **P4 — Long-term** | Informational + higher competition + builds brand awareness | Plan as pillar content; build supporting cluster first |

## Negative Keywords (For Paid Campaigns)

These keywords should be EXCLUDED from paid campaigns for SecurityBlogs and security clients:

- home security (unless targeting residential — usually too broad and consumer-focused)
- security guard jobs / security guard training (job-seeker intent, not buyer intent)
- security camera hack / how to hack security camera (wrong intent entirely)
- free security camera (giveaway-hunter intent)
- security camera reviews [specific consumer product names] (consumer B2C intent)
- cyber security degree / security course / security certification (education intent)
- national security / homeland security (wrong industry entirely)
