## Notes for Writers

- Australian English throughout (practise as verb, licence as noun, optimise, recognise, etc.)
- Reference specific Australian security publications where possible: Security Electronics & Networks (SEN), Security Insider, ASIAL's Security Solutions magazine
- Note that the Australian Press Council and ACMA govern media standards — relevant for crisis PR content
- Be careful distinguishing between cybersecurity PR (technology, data, hacking) and physical security PR (guards, CCTV, access control) — these reach different journalists and publications
- Avoid overpromising on earned media outcomes — PR is relationship-driven and results are never guaranteed
- The crisis PR content should be carefully positioned as guidance, not legal or professional advice
- Cross-link heavily to Cluster 04 (AI Visibility) — PR-driven brand mentions are increasingly important for AI citation signals
- Each supporting page: 1,000–2,000 words; pillar page: 3,500+ words; blog posts: 700–1,200 words
