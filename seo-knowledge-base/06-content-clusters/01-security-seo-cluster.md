## Notes for Writers

- All content should be written in **Australian English** (practise, optimise, colour, etc.)
- Cite Australian-specific examples: Sydney, Melbourne, Brisbane, Perth, Adelaide security markets
- Reference Australian directories: True Local, Yellow Pages Australia, Whereis, Hotfrog AU
- Mention Australian regulatory context where relevant (Australian Security Industry Association Limited — ASIAL)
- Each supporting page should be 1,500–2,500 words; blog posts 800–1,500 words; pillar page 4,000+ words
- Every piece must link back to the pillar page at least once
- Include original statistics, client data (anonymised), or curated industry stats where possible
