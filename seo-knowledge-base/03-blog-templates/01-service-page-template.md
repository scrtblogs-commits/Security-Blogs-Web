# [H1: PRIMARY KEYWORD — Written as a Clear, Benefit-Driven Headline]

ℹ️ *Section Note — H1:* The H1 must contain your primary keyword. Write it as a compelling statement of what you offer and who it's for. Keep it under 70 characters where possible. The keyword should appear near the start of the H1, not at the end.

- "Security Guard Services for Australian Businesses"
- "CCTV Installation & Monitoring Across Australia"

- "We Provide Security" ✗

## PART 4 — PRE-PUBLISH CHECKLIST

**SEO Fundamentals**
- [ ] Primary keyword in H1, meta title, meta description, first 100 words of body, at least one H2, and one image alt attribute
- [ ] Secondary keywords appear naturally in at least two H2 or H3 subheadings
- [ ] Meta title is 60 characters or fewer
- [ ] Meta description is 155 characters or fewer
- [ ] Canonical URL is set correctly

**Content Quality**
- [ ] All `[PLACEHOLDER]` values have been replaced with real content
- [ ] All Section Notes (ℹ️) have been deleted before publishing
- [ ] Word count is between 1,500 and 2,500 words
- [ ] Australian English used throughout (licence, organisation, specialised, authorised, etc.)
- [ ] All statistics are cited with source and date
- [ ] No fabricated statistics or unsourced claims

**Links**
- [ ] Minimum 3–5 internal links with descriptive keyword-rich anchor text
- [ ] External links to sources open in a new tab (`target="_blank" rel="noopener"`)
- [ ] No broken links (validate with a link checker before publishing)

**Technical**
- [ ] Both JSON-LD schemas added to `<head>` or via CMS schema plugin
- [ ] Schema validated via Google Rich Results Test — no errors
- [ ] All images have descriptive alt text (include keyword where it sounds natural)
- [ ] Page is mobile-friendly (test via Google Mobile-Friendly Test)
- [ ] Page load time under 3 seconds on mobile (test via PageSpeed Insights)

**Post-Publish**
- [ ] URL submitted to Google Search Console → URL Inspection → Request Indexing
- [ ] Page added to XML sitemap (auto or manual)
- [ ] At least 2–3 existing pages on the site updated to link to this new page
- [ ] Page added to SecurityBlogs internal content tracker spreadsheet
